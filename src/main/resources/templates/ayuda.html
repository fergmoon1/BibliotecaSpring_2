Ayuda

Esta es la página de ayuda para la aplicación de la biblioteca. Aquí tienes algunas instrucciones básicas:

- Para ver todos los elementos, visita el catálogo.

- Para agregar un nuevo elemento, usa la opción "Agregar Elemento" y selecciona el tipo (Libro, Revista o DVD).

- Usa el formulario de búsqueda para encontrar elementos por título o autor.

Si tienes problemas, verifica que los datos ingresados sean correctos y que la base de datos esté configurada adecuadamente.
